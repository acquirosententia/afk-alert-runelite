# AFK Alert
Alerts you when you've been idle too long so you don't get logged out mid-grind.
 
Fires a tray notification and keeps re-firing it on a configurable interval until you do something. The in-game overlay counts down your remaining time and blinks when the timer expires.
 
## Reset triggers
Pick whichever combination fits what you're doing:
- **Reset on XP drop** — good for AFK skilling (redwoods, etc.)
- **Reset on hitsplat received** — good for NMZ absorptions where you're not gaining XP
- **Reset on any player input** — catch-all for anything where you're clicking around (slayer, etc.)
